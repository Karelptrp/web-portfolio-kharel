Project Gallery

Prototype Fire Prevention Bell With Sensor

My first Project in 10th grade at shs was to make an early fire detection device using an Arduino Nano microcontroller and a 16x2 mini LCD display as a visual indicator and a buzzer as an indicator if there is a fire which is reported by an infrared sensor and a temperature sensor.

Smart Trash Can With Distance Detection Sensor

My second project this time with a team that was created to fulfill the final assignment of semester 2, at that time the issue of covid was hot, so we made an innovation of a trash can that can be opened without physical contact, namely with an ultrasonic distance sensor with an algorithm that opens when humans are less than 3 meters away, the lid quickly opens and closes 12 seconds after humans leave.

Gas Leak Detector and Room Monitoring Proto

a simple IOT project that I made, connecting the telegram and Whatsapp APIs to a tool that can detect gas leaks, this tool was made because I have a business that requires the use of a gas stove, this tool is not only able to detect gas leaks but also able to detect room temperature and humidity, connected to CCTV so that this tool package can provide real-time reports when there is an indication of a gas leak or a sudden increase in room temperature for fire prevention.
